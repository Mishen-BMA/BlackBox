About BlackBox

Your ultimate CTF companion

What is BlackBox?

BlackBox is an all-in-one browser-based CTF toolkit built for security researchers and CTF players. No installation needed - everything runs in your browser.

Built with pure HTML, CSS and JavaScript. No backend. Works offline.

Features

  • 50+ tools across 7 categories
  • Global search - find any tool instantly
  • Copy button on every output
  • Input history saved locally
  • 3 themes - Cyber, Hacker, Midnight
  • Keyboard shortcuts for power users
  • Fully responsive - works on mobile
  • 100% offline capable

Keyboard Shortcuts

  • Ctrl + Enter - Run current tool
  • / - Focus search bar
  • Esc - Go back to category
  • Ctrl + C - Copy output (when focused)
